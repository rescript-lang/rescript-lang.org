Use quoted field names inside `{ ... }` to define an object type.

### Example

<CodeTab labels={["ReScript", "JS Output"]}>

```res
type user = {"name": string, "age": int}

let u: user = {"name": "Ada", "age": 36}
```

```js
var u = {
  name: "Ada",
  age: 36,
};
```

</CodeTab>
