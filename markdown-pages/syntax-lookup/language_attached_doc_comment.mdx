Use an attached doc comment directly before a value or type declaration.

### Example

<CodeTab labels={["ReScript", "JS Output"]}>

```res
/** Returns the full name. */
let fullName = (first, last) => first ++ " " ++ last
```

```js
function fullName(first, last) {
  return first + " " + last;
}
```

</CodeTab>
