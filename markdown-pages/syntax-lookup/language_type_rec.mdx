Use `type rec` to define recursive types. Combine it with `and` for mutually recursive types.

### Example

<CodeTab labels={["ReScript", "JS Output"]}>

```res
type rec expr =
  | Int(int)
  | Add(expr, expr)
and node = {
  value: expr,
  next: option<node>,
}
```
