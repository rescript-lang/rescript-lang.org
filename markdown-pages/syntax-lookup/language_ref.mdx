Use `ref(value)` to create a mutable reference cell. Read via `.contents` and update via `:=`.

### Example

<CodeTab labels={["ReScript", "JS Output"]}>

```res
let counter = ref(0)

counter := counter.contents + 1
```

```js
var counter = {
  contents: 0,
};

counter.contents = (counter.contents + 1) | 0;
```

</CodeTab>
