Use `{.}` to describe an object type with no known fields.

### Example

<CodeTab labels={["ReScript", "JS Output"]}>

```res
type empty = {.}

let acceptEmpty = (_value: empty) => ()
```

```js
function acceptEmpty(_value) {}
```
