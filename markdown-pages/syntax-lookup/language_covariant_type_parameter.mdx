Use a `+` variance annotation on a type parameter to mark it as covariant.

### Example

<CodeTab labels={["ReScript", "JS Output"]}>

```res
type source<+'a> = unit => 'a

let fromValue = (x: 'a): source<'a> => () => x
```

```js
function fromValue(x) {
  return (_) => x;
}
```

</CodeTab>
