Use `let rec` to define recursive functions.

### Example

<CodeTab labels={["ReScript", "JS Output"]}>

```res
let rec factorial = n =>
  if n <= 1 {
    1
  } else {
    n * factorial(n - 1)
  }
```

```js
function factorial(n) {
  if (n <= 1) {
    return 1;
  } else {
    return Math.imul(n, factorial((n - 1) | 0));
  }
}
```
