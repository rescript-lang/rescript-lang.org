Use `/* ... */` for multi-line comments.

### Example

<CodeTab labels={["ReScript", "JS Output"]}>

```res
/*
  This is a block comment.
*/
let greeting = "hello"
```

```js
var greeting = "hello";
```

</CodeTab>
