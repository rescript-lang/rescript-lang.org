Use `<MyComponent ... />` to render a component in JSX.

### Example

<CodeTab labels={["ReScript", "JS Output"]}>

```res
@react.component
let make = (~name) => <h1> {React.string("Hello " ++ name)} </h1>

let view = <make name="ReScript" />
```

```js
import * as React from "react";

function Playground(props) {
  return React.createElement("h1", undefined, "Hello " + props.name);
}

var make = Playground;

var view = React.createElement(Playground, {
  name: "ReScript",
});
```
