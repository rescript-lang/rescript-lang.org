Use `exception` to declare custom exception constructors.

### Example

<CodeTab labels={["ReScript", "JS Output"]}>

```res
exception NotFound(string)

let message = try {
  throw(NotFound("Missing value"))
} catch {
| NotFound(text) => text
}
```

```js
var message = "Missing value";
```

</CodeTab>
