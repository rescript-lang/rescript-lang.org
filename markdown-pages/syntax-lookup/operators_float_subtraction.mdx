> Deprecated, you can just use `-` for both integers and floats since v12.

This operator performs _floating point_ subtraction.

<CodeTab labels={["ReScript", "JS Output"]}>

```res
let result = 3.0 -. 2.5
```

```js
var result = 3.0 - 2.5;
```

</CodeTab>

For subtracting _integers_ see the `-` operator.
