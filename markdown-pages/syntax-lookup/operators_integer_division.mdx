This operator performs _integer_ division, with the result truncated to an integer value.

If the second argument is _zero_ then a `Division_by_zero` exception is thrown. Read about [exception](../docs/manual/exception.mdx)

<CodeTab labels={["ReScript", "JS Output"]}>

```res
let result = 3 / 2
```

```js
var result = 1;
```

</CodeTab>
