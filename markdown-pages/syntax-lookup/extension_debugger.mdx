`%debugger` is used to insert a JavaScript `debugger` statement.

<CodeTab labels={["ReScript", "JS Output"]}>

```res
let f = (x, y) => {
  %debugger
  x + y
}
```

```js
function f(x, y) {
  debugger;
  return (x + y) | 0;
}
```

</CodeTab>
