Use `?` on a record field to mark it optional.

### Example

<CodeTab labels={["ReScript", "JS Output"]}>

```res
type user = {
  age: int,
  name?: string,
}

let withName: user = {age: 5, name: "Ada"}
let withoutName: user = {age: 7}
```

```js
var withName = {
  age: 5,
  name: "Ada",
};

var withoutName = {
  age: 7,
};
```

</CodeTab>
