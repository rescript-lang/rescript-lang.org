Use `|` inside a pattern to match multiple cases with one branch.

### Example

<CodeTab labels={["ReScript", "JS Output"]}>

```res
let n = 1

let label = switch n {
| 0 | 1 => "small"
| _ => "other"
}
```

```js
var n = 1;

var label = n === 0 || n === 1 ? "small" : "other";
```

</CodeTab>
