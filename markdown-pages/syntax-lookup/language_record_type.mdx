A record type defines a fixed structure with named fields.

### Example

<CodeTab labels={["ReScript", "JS Output"]}>

```res
type myRecord = {
  name: string,
  age: int,
}

let myRecord: myRecord = {name: "Ada", age: 36}

let label = switch myRecord {
| {name, age} => name ++ " (" ++ Int.toString(age) ++ ")"
}
```

```js
var myRecord = {
  name: "Ada",
  age: 36,
};

var label = "Ada (36)";
```

</CodeTab>
