> Available in v12+

You write regular expressions in ReScript just like you do in JavaScript.

<CodeTab labels={["ReScript", "JS Output"]}>

```res
let regex = /^hello/
let result = regex->RegExp.test("hello world")
```

```js
let regex = /^hello/;

let result = regex.test("hello world");
```

</CodeTab>
