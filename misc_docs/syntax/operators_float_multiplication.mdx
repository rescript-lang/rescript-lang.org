This operator performs _floating point_ multiplication.

<CodeTab labels={["ReScript", "JS Output"]}>

```res
let result = 1.5 *. 2.3
```

```js
var result = 1.5 * 2.3;
```

</CodeTab>

For multiplying _integers_ see the `*` operator.
