This operator performs _floating point_ division.

<CodeTab labels={["ReScript", "JS Output"]}>

```res
let result = 3.0 /. 2.5
```

```js
var result = 3.0 / 2.5
```

</CodeTab>

For dividing _integers_ see the `/` operator.
