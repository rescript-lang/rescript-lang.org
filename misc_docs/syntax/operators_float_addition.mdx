This operator performs _floating point_ addition.

<CodeTab labels={["ReScript", "JS Output"]}>

```res
let result = 1.3 +. 0.5
```

```js
var result = 1.3 + 0.5;
```

</CodeTab>

For adding _integers_ see the [`+`](#integer-addition) operator.
